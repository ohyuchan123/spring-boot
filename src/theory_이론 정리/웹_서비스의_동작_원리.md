# 웹 서비스의 동작 원리

```
웹 서비스는 client와 server에 요청과 응답으로 동작합니다

client : 서비스를 사용하는 프로그램 또는 컴퓨터를 의미
server : 서비스를 제공하는 프로그램 또는 컴퓨터를 의미

resources에서 hello.html를 통해 chrome에 'Hello-spring'을 띄운 것 또한
client와 server의 요청과 응답의 결과이다.

browser : client로서 동작 , spring-boot : server의 역할을 수행 하였다
```

### localhost : 9090
```
localhost : 서버의 주소를 의미(내 컴퓨터의 주소)
8080 : 포트 번호(9090방에 요청) 

즉 localhost:9090에 spring-boot가 동작하고 있다 
```
